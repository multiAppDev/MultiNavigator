## Short description (up to 255 characters)
A clean, fast, and truly private Windows browser. No ads, no telemetry, no accounts required. Open source since 2001. View 4 websites simultaneously with real isolated incognito tabs.

---

## Long description

MultiNavigator is a free, open source Windows browser built on Microsoft WebView2, designed for users who want a fast, clean, and genuinely private browsing experience — with no compromises.

**A browser with history. 25 years of it.**
MultiNavigator was first released in 2001 — before Firefox, before Chrome. It was one of the first browsers to let you view multiple web pages simultaneously on a single screen. 25 years later, version 7.0.0 brings that same vision back, completely rebuilt from the ground up for modern Windows. Not a single line of the original code remains — same soul, new foundation.

---

**What makes MultiNavigator different?**

🔒 Zero telemetry — MultiNavigator collects absolutely nothing about you. No usage data, no crash reports, no analytics sent anywhere. Ever.

🪟 4 simultaneous browsers in split view — View four independent web pages at the same time on a single screen. Drag the central divider to resize each pane exactly how you want it.

🕵️ True isolated incognito tabs — Run private and normal browsing sessions side by side in the same window, completely sandboxed from each other. A unique approach not found in mainstream browsers.

🎛️ Granular privacy control — Decide individually what to allow on the web: telemetry, trackers, fingerprinting, cookies and more. Not just an on/off switch — real control.

🎨 Fully customizable themes — Choose from multiple built-in themes or design your own with the integrated theme editor. Make the browser truly yours.

🏠 Independent start page per browser — Configure a different home page for each of the four browser panes.

🔗 Open links in any browser — Right-click any link and open it directly in Chrome, Firefox, Edge, or any other browser installed on your system.

🖱️ Drag tabs to other browsers — Move a tab directly from MultiNavigator into another browser window with a simple drag.

📝 Notes with startup alerts — Attach notes to your browser that remind you of important things every time you open it.

📦 Import from other browsers — Migrate your bookmarks and settings directly from Chrome, Firefox, Edge and others.

💾 Session restore — MultiNavigator automatically saves and restores your last session so you never lose your place.

🔍 Configurable search engine — Choose your preferred search engine.

All your data — history, bookmarks, notes, settings — stays on your device in local files. Nothing is ever sent to any server.

**Privacy note regarding WebView2.**
MultiNavigator is built on Microsoft WebView2, which uses the Chromium engine for rendering web pages. The developer has no access to data processed by WebView2 or Chromium. For full details, please read the Privacy Policy and Disclaimer included with the app.
