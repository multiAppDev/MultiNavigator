private int GetDropIndexByPosition(ItemsControl panel, Point mousePosInPanel)
{
    int count = panel.Items.Count;
    if (count == 0) return 0;

    double x = 0;

    for (int i = 0; i < count; i++)
    {
        var container = panel.ItemContainerGenerator.ContainerFromIndex(i) as FrameworkElement;
        if (container == null) continue;

        double width = container.ActualWidth;
        double center = x + width / 2;

        if (mousePosInPanel.X < center)
            return i;

        x += width;
    }

    return count;
}

El problema está en la animación de 1500ms (línea 1269). Eso es 1.5 segundos de animación cada vez que cambia algo. Durante el drag, esto causa que:
Se inicie una animación de 1.5s
Mientras anima, mueves el ratón
Se cancela y empieza otra animación de 1.5s
Conflicto de animaciones = parpadeo

creo que cuando se está haciendo drag de una pestaña, no se puede activar drag en ninguna otra.

Solo hace este efecto cuando estoy hacidno drag, y la pestaña que muevo y la que se esta moviendo coinciden en algun punto con el puntero del ratón y lo muevo
